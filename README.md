# Tutor_App
Application for tuition Classes

Making a simple application that can help teachers of coaching classes, students and parents to keep a track on development of student.

It will have 3 faces:
	1. Admin/Teacher
	2. Student
	3. Parent

Functionalities:
	1. Exam results with history
	2. Calendar for upcoming exams
	3. Attendance Sheet
	4. Adding leave
	5. Sending email to parents if marked as leave
	6. Achievements
	7. Reminders and Notices
